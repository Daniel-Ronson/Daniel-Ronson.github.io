Music Microservies

Overview:

  • Create and generate playlists with REST microservices.
  • Services for Playlists, Tracks, Descriptions, Users, Playlist Generator
  • Backend web app that could be used for a music sharing website

Technology:

  • Built with Flask web framework using Python,  ScyllaDB,  Kong(NGINX),   Memecached,   Minio
  • Source Code

Additional Features:

  • Generate XSPF playlist; an XML format playlist that can be imported to a media player
  • MP3 storage
  • Create playlist with desired tracks
  • Each service implements GET,POST,PUT,DELETE
  • Load balancer and horizontally scalable infrastructure

Data Flow:

  1. Create user account
  2. Create playlist by adding tracks
  3. Generate playlist in XML format
  4. Import playlist to VLC media player
